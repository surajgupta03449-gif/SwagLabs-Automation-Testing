# Swag Labs E-Commerce Testing Project (Manual + Automation)

## Project Overview

This project demonstrates end-to-end testing of the Swag Labs e-commerce application using both Manual Testing and Selenium Automation Testing.

The project covers the complete software testing lifecycle, including test planning, test scenarios, manual test cases, automation scripts, execution reports, screenshots, and documentation.

---

## Application Under Test

* **Application:** Swag Labs
* **Application Type:** E-Commerce Web Application
* **URL:** https://www.saucedemo.com/

---

## Project Objectives

* Validate the functionality of the Swag Labs application.
* Design and execute manual test cases.
* Automate repetitive test scenarios using Selenium WebDriver.
* Generate execution reports using Extent Reports and TestNG.
* Demonstrate Page Object Model (POM) framework implementation.

## How to Execute

1. Clone the repository.
2. Import the Selenium_Automation project into Eclipse.
3. Update Maven dependencies.
4. Execute the `testng.xml` file.
5. View TestNG and Extent Reports.
