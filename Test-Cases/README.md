# Project Description

This project demonstrates comprehensive quality assurance for the Swag Labs e-commerce web application through both manual testing and Selenium automation testing.

The project includes detailed manual test scenarios, manual test cases, automation scripts, execution reports, requirement traceability, defect tracking, and automation coverage mapping.

The automation framework is developed using Java, Selenium WebDriver, TestNG, Maven, and the Page Object Model (POM) design pattern.

---

# Project Objectives

* Verify the functionality of the Swag Labs application.
* Create comprehensive manual test scenarios and detailed test cases.
* Automate repetitive functional test cases using Selenium WebDriver.
* Generate execution reports using TestNG and Extent Reports.
* Demonstrate industry-standard QA documentation.
* Maintain traceability between manual and automated test cases.
